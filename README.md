### 一键导出Excel文件
##### 通过对当前页面的Table下的整理，将其拼接成一个总Table，并按要求进行表头位置的整体，当error比大于0%时，字体颜色红色
> #### 注意：本一键导出Excel功能只适应本公司页面，针对其他页面不适应，导出一个空的Excel
